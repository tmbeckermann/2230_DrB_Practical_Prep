Put practical identification images in this folder.

Use the practical key number at the start of the filename:

- 11.jpg
- 11-closeup.jpg
- 12.png
- item-13.webp
- practical-20.jpg

After adding or replacing images, rebuild the site with `build_student_site.py`.
Images with matching numbers appear in the Numbered Key table and create an
Image practical ID drill deck automatically. Multiple images can share the same
number when the number comes first in each filename.
